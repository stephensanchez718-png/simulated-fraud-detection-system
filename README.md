
# Simulated Fraud Detection System

## Overview
Python-based SOC fraud detection simulation that analyzes suspicious banking behavior using automated alert scoring and anomaly detection.

## Features
- Failed login detection
- VPN/suspicious IP detection
- Transaction spike analysis
- Impossible travel detection
- New device detection
- Severity scoring system
- Automated SOC-style alerts

## Technologies Used
- Python
- Jupyter Notebook
- Randomized data simulation
- Security event analysis

## Example Alert Output

```text
[SOC ALERT]
Severity: CRITICAL
Issues: Failed login attempts, VPN access, transaction spike
Action: Investigate / MFA / Possible Lock
```

## Project Purpose
This project simulates how a Security Operations Center (SOC) identifies potentially malicious account activity using behavioral analysis and automated detection logic.
